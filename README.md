## Установка и запуск

**Требования:** Node.js версии **20.19.0** или выше.

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите проект в режиме разработки (запустит моковый сервер и данные будет брать с него - npm run dev:server + dev:client):
   ```bash
   npm run dev
   ```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Команды

В проекте настроены следующие скрипты:

- `npm run dev` — запуск сервера для разработки на Webpack Dev Server + Express.
- `npm run build` — сборка проекта для продакшена.
- `npm run type-check` — проверка типов TypeScript.
- `npm run lint` — проверка кода линтером ESLint.
- `npm run lint:fix` — форматирование кода с помощью ESLint.
- `npm run format` — форматирование кода с помощью Prettier.
- `npm run clean` — удаление папки `dist`.

## Технологический стек
- **Node >=20.19.0 - минимально рекомендуемая версия Node**
- **React 19** — библиотека для создания пользовательских интерфейсов.
- **TypeScript** — строгая типизация для повышения надежности кода.
- **Webpack 5** — сборщик модулей с кастомной конфигурацией.
- **GSAP** — библиотека для продвинутых анимаций.
- **Swiper** — современный слайдер для мобильных и десктопных устройств.
- **Sass (SCSS)** — препроцессор для стилизации.
- **Feature-Sliced Design (FSD)** — архитектурная методология проектирования.

## Структура проекта

Проект организован по методологии **FSD**:

- `src/app`: Инициализация приложения (стили, провайдеры).
- `src/widgets`: Крупные самостоятельные блоки интерфейса (например, Slider).
- `src/shared`: Переиспользуемый код, не привязанный к бизнес-логике (типы, UI-компоненты).

### Требования к реализации (ТЗ)

1. Необходимо использовать Typescript
2. Можно использовать React.js или нативный JS на Ваш выбор
3. В случае использования нативного JS, можно воспользоваться любым удобным для Вас HTML-шаблонизатором или обычным HTML
4. Стилизация с использованием SASS/SCSS (В случае работы с React.js, возможно использование styled-components)
5. Сборка проекта с помощью Webpack
6. Для работы со слайдерами необходимо использовать библиотеку Swiper
7. Для реализации js-анимаций можно использовать библиотеку gsap

---
- Не использовать JQuery
- Не использовать Bootstrap, Tailwind и т.п.
- Не использовать библиотеки с готовыми UI-компонентами такие, как MaterialUI, AntDesign и т.п.
